## Qubes-OS 工具翻译及说明

### Qubes Menu settings
**Qubes 菜单设置**：用于配置和管理 Qubes OS 中的菜单选项和显示设置，用户可以自定义菜单的外观和功能。

### Qubes Policy Editor
**Qubes 策略编辑器**：一个工具，用于创建和管理 Qubes OS 中的安全策略。用户可以通过它设置虚拟机之间的访问权限和交互规则，以增强系统的安全性。

### Qubes Template Manager
**Qubes 模板管理器**：用于管理和配置 Qubes OS 中的模板。用户可以通过它创建、删除和更新模板，以便为虚拟机提供基础操作系统和应用程序环境。

### Qubes Template Switcher
**Qubes 模板切换器**：允许用户在不同的模板之间切换，以便为现有的虚拟机更改基础模板。这对于更新或更改虚拟机的操作系统版本非常有用。

### Qubes Update
**Qubes 更新**：用于更新 Qubes OS 及其虚拟机的工具，确保系统和应用程序保持最新状态。

### Restore Backup
**恢复备份**：用于从先前创建的备份中恢复虚拟机或系统状态的功能，确保数据和配置的安全。

### Backup Qubes
**备份 Qubes**：允许用户创建虚拟机的备份，以保护重要数据和设置，便于后续恢复。

### Create New Qube
**创建新 Qube**：用于创建新的虚拟机（Qube），用户可以根据需要配置其操作系统和应用程序。

### Create Qubes VM
**创建 Qubes 虚拟机**：与“创建新 Qube”类似，允许用户创建新的虚拟机实例，以便在 Qubes OS 中运行应用程序和服务。

### Qubes Manager
**Qubes 管理器**：一个图形用户界面工具，用于管理虚拟机、模板和其他 Qubes 资源，用户可以通过它创建、删除和配置虚拟机。

这些工具和功能在 Qubes OS 中帮助用户有效地管理和维护虚拟机及其数据。
