<summary>解压与压缩（点击展开）</summary>

* 压缩内含多个 文件夹 及 文件 ,将其浓缩在一个 格式 的文件里.
  * 什么是格式?  以 文件 1145.zip 为例:
  * 1145 为 文件名字
  * .zip为格式
  * 常见 压缩格式 例如.zip,rar,7z,tar,tar.gz,xz
* 例外:
  * 分盘格式为 常见格式 + .001(循环)
  * 例如:1145.zip.001 和 1145.zip.002
  * 两个文件在 同一文件夹 下 才是 一个完整的压缩包
  * 这两个只算作 一个压缩包
  * 只需解压其中一个
* 文件夹是什么?
  * 指可以放多个 文件 的工具包
* 文件 格式是什么?
  * 前面提到了压缩格式
  * 格式分为很多种类,这里就不细分了,请上网搜索
  * 这里只列举 windows 常见软件格式
  * 格式为 XX.exe

<summary>如何使用鼠标（点击展开）</summary>

* 常见鼠标
* 左键为 点击 操作
* 右键为 选择 操作
* 中间的 滚轮 为 上下翻转 操作
* 例外,功能键(可自定义),以下为默认功能
  * 如侧边  上边 的键为 返回下一步 操作
  * 如侧边  下边 的键为 返回上一步 操作
  * 如 中间小按键 为调整 dpi 鼠标的灵敏度

<details>
<summary>如何使用键盘（点击展开）</summary>

* 上下左右 --> WSAD --> 右边的 ↑ ↓← →
* 功能键
  * F11全屏,F12查看页面源代码(非实际源码).
  * F5刷新,ESC退出,PS截图全屏.
* 组合功能键(+是一起按的意思)
  * 关闭窗口  -->  CTRL + W -->  ALT + F4
  * 复制粘贴  -->  CTRL + C -->  CTRL + V
  * 选择全部  -->  CTRL + A
  * 剪切  -->  CTRL + X
  * 撤回操作  -->  CTRL + Z
  * 搜索操作  -->  CTRL + F
