<details>
<summary>硬件组成部分（点击展开）</summary>

* 五大组成部分: 控制器、运算器、存储器、输入设备和输出设备。
* 控制器: 计算机的神经中枢，负责指挥计算机各个部件协调工作.
* 运算器: 运算器负责执行各种算术和逻辑运算，是计算机的逻辑大脑.
* 存储器: 存储程序、数据和各种信号.分为内存和外存.
  * 内存用于临时存储数据，速度快但断电后数据会丢失.
  * 外存用于永久存储数据，速度较慢但断电后数据不会丢失.
  * 也就是 内存条 和 硬盘,
  * 硬盘又分为 固态硬盘 与 机械硬盘 .
* 输入设备:向计算机输入信息,如 键盘、鼠标、扫描仪、触摸屏 等.
* 输出设备:用于接收计算机的数据输出,如 显示器、打印机、音响 等.
* 例外:
  * CPU 是冯·诺依曼五大组成部分中运算器和控制器的具体物理实体，是计算机的“大脑”。
  * 主板 是支撑和连接所有五大组成部分的物理平台和通信枢纽，是计算机的“躯干和骨架”。
  * CPU 是核心处理部件本身，而主板是承载并连接所有部件的基石。
  * GPU 是与CPU并列的核心硬件计算单元，通过主板与系统其他部分连接。
  * GPU 已经从“图形处理器”演变为“通用图形处理器”，并进一步成为通用并行计算加速器，是驱动人工智能、科学发现和视觉体验革命的核心引擎。
